SQL简介

SQL的通用语法

  1. SQL 语句可以单行或多行书写，以分号结尾。

  2. MySQL 数据库的 SQL 语句不区分大小写，关键字建议使用大写（看个人习惯）。

  3. 注释

    • 单行注释：– 注释内容 或 #注释内容（MySQL 特有）

    • 多行注释：/* 注释 */

SQL语言分类

  • DDL 数据定义语言，操作数据仓库，表等
  • DML 数据操作语言，对表中的数据进行增删改
  • DQL 数据查询语言，对表中的数据进行查询
  • DCL 数据控制语言，对数据库进行权限控制

分组查询（group by）

  1. 分组查询语法

  • where 和 having 区别：
    • 执行时机不一样：where 是分组之前进行限定，不满足where条件，则不参与分组，而having是分组之后对结果进行过滤。
    • 可判断的条件不一样：where 不能对聚合函数进行判断，having可以。
  • 执行顺序：==where > 聚合函数 > having==

  1. 分页查询语法

  • 计算公式：==起始索引 = （档前页码-1）* 每页显示的条目数==
  • tips：
    • 分页查询limit是MySQL的数据库方言
    • Oracle分页查询使用rownumber
    • SQL Server分页查询使用top
